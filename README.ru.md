# Приложение Фонарик

Простое приложение фонарика на базе Flutter, разработанное с желтой банановой темой. Это приложение позволяет пользователям включать вспышку камеры в качестве источника света, с интуитивным интерфейсом и поддержкой нескольких языков.

## Основные Функции

- **Управление Фонариком**: Кнопка питания для включения/выключения вспышки камеры.
- **Динамические Анимации**: Плавные переходы с эффектами свечения и градиента при включенном свете.
- **Многоязычность**: Поддержка 6 языков: Индонезийский, Английский, Мандарин, Японский, Русский и Арабский.
- **Реклама AdMob**: Интеграция межстраничной рекламы при выходе из приложения для монетизации.
- **Адаптивность**: Дизайн, адаптирующийся к различным размерам экранов и виртуальным кнопкам навигации.
- **Экран Загрузки**: Начальный экран с иконкой OpenMoji.

## Системные Требования

- Flutter SDK: ^3.10.4
- Dart SDK: ^3.0.0
- Android: minSdkVersion 21 (Android 5.0)
- iOS: Deployment target 11.0 (если требуется)

## Настройка Приложения

### 1. Клонировать Репозиторий

```bash
git clone https://github.com/username/senter.git
cd senter
```

### 2. Установить Зависимости

Убедитесь, что Flutter установлен и путь настроен.

```bash
flutter pub get
```

### 3. Настроить AdMob (Опционально, для Монетизации)

- Зарегистрируйтесь в AdMob на [admob.google.com](https://admob.google.com).
- Создайте приложение и межстраничный рекламный блок.
- Обновите ID приложения и рекламного блока в следующих файлах:
  - `android/app/src/main/AndroidManifest.xml`: Замените `android:value` на ваш AdMob App ID.
  - `lib/main.dart`: Замените `adUnitId` в `_loadInterstitialAd()` на ваш межстраничный Unit ID.

### 4. Генерировать Локализацию (Если Необходимо)

Если есть изменения в ARB файлах в `lib/l10n/`, регенерируйте:

```bash
flutter pub run intl_utils:generate
```

### 5. Собрать для Android

```bash
flutter build apk --release
```

Или для отладки:

```bash
flutter build apk --debug
```
